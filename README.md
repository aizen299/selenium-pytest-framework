# Selenium Pytest Automation Framework

Enterprise-style Selenium automation framework built using Python and Pytest.

## Features
- Page Object Model (POM)
- Parallel execution with pytest-xdist
- Controlled slow mode for visual testing
- Screenshot capture on failure
- HTML test reports
- CI-ready GitHub Actions workflow

## Tech Stack
- Python
- Selenium WebDriver
- Pytest
- Pytest-xdist
- Pytest-html

source venv/bin/activate
